# OCSOrange

This Android(Kotlin) application is developed as part of a technical test for OCS Orange. 

Tech used:
- Clean Architecture
- MVVM as design pattern
- Retrofit for API calls
- Hilt for dependency injection
- AndroidX
- Navigation component
- Glide
- ExoPlayer
- Espresso
- Mockk
